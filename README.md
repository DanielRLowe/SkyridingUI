# SkyridingUI

A modern, customizable UI for tracking Skyriding speed and vigor charges in World of Warcraft.

## Features

- **Multiple Display Modes**: Choose from Horizontal bars, Speedometer, Circular display, or Vigor Orbs
- **Customizable Colors**: Full color customization for charge bars, arcs, and orb effects
- **Visibility Options**: Fine-grained control over which UI elements are displayed
- **Auto-hide Options**: Hide UI when grounded with full charges or hide speed displays when landed
- **Movable & Scalable**: Drag to position anywhere on screen, adjustable scale
- **Background Options**: Optional semi-transparent background for better visibility

## Display Modes

### Horizontal Bars
Classic bar-style display showing speed, acceleration, charges, and buffs in horizontal bars.

### Speedometer
Circular gauge with charge indicators arranged around the perimeter.

### Circular
Compact circular display with speed ring, charge arcs, and visual effects.

### Vigor Orbs
Gemstone orbs with animated energy effects, custom colors, and charging animations.

## Installation

### Via Addon Manager
- **CurseForge App**: Search for "SkyridingUI"
- **Wago Addons**: Search for "SkyridingUI"

### Manual Installation
1. Download the latest release
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or reload UI (`/reload`)

## Usage

- Type `/sui` or `/skyriding` to open options
- Click the minimap button to toggle lock/unlock
- Drag the UI when unlocked to reposition

## Options

Access the full options panel via:
- `/sui options`
- Minimap button right-click
- ESC > Interface > AddOns > Skyriding UI
